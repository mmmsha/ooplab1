# Лабораторная работа 1  
## Система управления курсами и преподавателями

Этот проект реализует систему управления учебными курсами университета.

Система позволяет:
- добавлять и удалять курсы
- назначать преподавателей на курсы
- записывать студентов на курсы
- получать список студентов конкретного курса
- получать список курсов конкретного преподавателя
- поддерживать различные типы курсов (онлайн и оффлайн)
